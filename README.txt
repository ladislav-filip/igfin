Plugin pro vedení finančního deníku

== Description ==

Tento plugin poskytuje funkce pro vedení finančního deníku - portfólia akcií.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `igfin.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
